Challenge ONE Iniciante em Programação

Contrua um decodificador de texto com Javascript

O Desafio

Trabalhar em uma aplicação que criptografa textos, assim você poderá trocar mensagens secretas com outras pessoas que saibam o segredo da criptografia utilizada.

Requisitos

Deve funcionar apenas com letras minúsculas, não pode utilizar letras com acentos nem caracteres especiais e deve ser possível converter uma palavra para a versão criptografada e também retornar uma palavra criptografada para a versão original

Cards no Trello

Será Disponibilizado um quadro no Trello que vai ser utilizado como ferramenta para gerenciar o desenvolvimento do projeto
